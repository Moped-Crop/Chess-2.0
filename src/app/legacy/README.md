# Legacy styles archive

`index.legacy.css` is a snapshot of `src/app/index.css` as of **2026-07-22**,
taken right before the Phase 6 redesign (deep-sapphire accent, UI primitives,
lucide icons). It is **not imported anywhere** — it lives here only as a
reference so an old rule can be looked up without digging through git.

The live, current stylesheet is `src/app/index.css`. A full pre-redesign
snapshot of the whole project also exists as the git tag `pre-redesign`.
